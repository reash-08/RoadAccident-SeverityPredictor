📌 Project Overview

The Smart City Road Accident Severity Prediction System is a Machine Learning powered web application designed to predict accident severity levels using traffic, environmental, and roadway conditions. The application helps identify potential accident risk levels and classifies incidents into Slight, Serious, or Fatal categories. The system combines Machine Learning, Flask Backend Development, SQLite Database Integration, and Interactive Data Visualization into a single platform.
