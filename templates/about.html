📌 Project Overview

The Smart City Road Accident Severity Prediction System is a Machine Learning based web application developed to predict accident severity using road, weather, traffic, environmental, and casualty information.


The application assists authorities, road safety analysts, and city planners in identifying accident risks, improving road safety measures, and supporting faster decision-making through intelligent severity prediction.

📂 Dataset Information

  • Dataset: UK Road Accident Dataset
  • Records: 50,000+
  • Target Variable: Accident Severity
  • Classes: Slight, Serious, Fatal
  • Prediction Type: Multi-Class Classification
