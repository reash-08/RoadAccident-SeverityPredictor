📌 Project Overview

The Smart City Road Accident Severity Prediction System is an intelligent web-based application designed to predict accident severity using historical road accident data and machine learning techniques.


By analyzing accident-related factors such as road conditions, weather, lighting, speed limits, and casualty information, the system provides real-time severity predictions that can assist decision-makers in improving road safety and emergency response planning.

🌍 Why This Project Matters

Road accidents are a major public safety concern and often result in injuries, fatalities, and economic losses. Early identification of accident severity can help authorities allocate resources more efficiently and improve response times.


This project demonstrates how machine learning can transform historical accident data into actionable insights, supporting safer transportation systems and smarter urban planning initiatives.

🎯 Project Objectives

  • Predict accident severity using historical accident records.
  • Provide instant accident risk assessment.
  • Support road safety analysis and decision-making.
  • Maintain historical prediction records for future review.
  • Visualize accident trends through an analytics dashboard.

📂 Dataset Information

  • Dataset: UK Road Accident Dataset
  • Records Used: 50,000+
  • Target Variable: Accident Severity
  • Severity Classes: Slight, Serious, Fatal
  • Prediction Type: Multi-Class Classification
